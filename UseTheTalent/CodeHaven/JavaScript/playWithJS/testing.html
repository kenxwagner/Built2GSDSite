Given the numeric series of f(n) = f(n-1) + f(n-2) starting with 0,1: 0, 1, 1, 2, 3, 5, 8, 13 … f(0) = 0 f(1) = 1 f(2) = 1 f(3) = 2 f(4) = 3 f(5) = 5 etc Below are 4 common ways of finding f(n): 1. Static Array static int fibarray[1000] = { 0,1,1,2,3,5,8,13,21 ... }; int fib(int n) { return fibarray[n]; } 3. Iterative int fib(int n) { if (n <= 1) return n; int a = 0, b = 1, c, i; for (i = 2; i <= n; i++) { c = a + b; a = b; b = c; } return b; } 2. Recursive int fib(int n) { if (n <= 1) return n; return fib(n-1) + fib(n-2); } 4. Pure Math int fib(int n) { float a = ((1 + √5)/2)^n); float b = ((1 - √5)/2)^n); return (int) ( 1/√5 * ( a - b ) ); }
