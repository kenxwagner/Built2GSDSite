Welcome to the world, ASP.NET Core!

Why yes, it is a single-page application, built with:

  • ASP.NET Core and C#
  • Angular 2 and TypeScript
  • Webpack
  • Bootstrap

What else is in there?

  • Client-side navigation. For example, click Counter then Back to return here.
  • Something needs to go in here to show Angular
  • Server-side prerendering. Angular 2 is prerendered on the server. What you get to see is the HTML for the client-side.
  • Webpack dev middleware. In development mode, there's no need to run the webpack build tool. Your client-side resources are dynamically built on demand. Updates are available as soon as you modify any file.
  • Webpack is a great deal of help
  • Hot module saves me time with waiting for changes. The new instance gets injected quickly.
